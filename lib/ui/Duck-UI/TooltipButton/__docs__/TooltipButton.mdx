# TooltipButton

The `TooltipButton` component is a button that can optionally display a tooltip when collapsed. 

## Usage

### Default TooltipButton

```jsx
import { TooltipButton } from '../TooltipButton';
import { TooltipProvider } from '@/ui/ShadcnUI';
import { Icon } from '@/assets/svgs';

<TooltipProvider>
  <TooltipButton
    button={{
      title: 'Inbox',
      label: '12',
      icon: Icon.inbox,
      onClick: () => alert('Button clicked!'),
    }}
    isCollapsed={false}
  />
</TooltipProvider>;
```

### Collapsed TooltipButton

```jsx
import { TooltipButton } from '../TooltipButton';
import { TooltipProvider } from '@/ui/ShadcnUI';
import { Icon } from '@/assets/svgs';

<TooltipProvider>
  <TooltipButton
    button={{
      title: 'Inbox',
      label: '12',
      icon: Icon.inbox,
      onClick: () => alert('Button clicked!'),
    }}
    isCollapsed={true}
  />
</TooltipProvider>;
```
