## Usage

```tsx
import {
  Command,
  CommandDialog,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
  CommandSeparator,
  CommandShortcut,
} from '@/components/ui'
```

```tsx
<Command>
  <CommandInput placeholder="Type a command or search..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Suggestions">
      <CommandItem>Search GitHub</CommandItem>
      <CommandItem>Search Twitter</CommandItem>
      <CommandItem>Search Discord</CommandItem>
    </CommandGroup>
    <CommandSeparator />
    <CommandGroup heading="Settings">
      <CommandItem>General</CommandItem>
      <CommandItem>Profile</CommandItem>
      <CommandItem>Notifications</CommandItem>
    </CommandGroup>
  </CommandList>
</Command>
```

### Dialog

<ComponentPreview name="CommandDialogDemo" description="A component dialog." showSettings={false} />

To show the command menu in a dialog, use the `<CommandDialog />` component, or use the `<Button/>` component with the `command` variant.

```tsx
export function CommandMenu() {
  const [open, setOpen] = React.useState(false)

  React.useEffect(() => {
    const down = (e: KeyboardEvent) => {
      if (e.key === 'k' && (e.metaKey || e.ctrlKey)) {
        e.preventDefault()
        setOpen(open => !open)
      }
    }
    document.addEventListener('keydown', down)
    return () => document.removeEventListener('keydown', down)
  }, [])

  return (
    <CommandDialog
      open={open}
      onOpenChange={setOpen}
    >
      <CommandInput placeholder="Type a command or search..." />
      <CommandList>
        <CommandEmpty>No results found.</CommandEmpty>
        <CommandGroup heading="Suggestions">
          <CommandItem>Settings</CommandItem>
          <CommandItem>Messages</CommandItem>
          <CommandItem>Search</CommandItem>
        </CommandGroup>
      </CommandList>
    </CommandDialog>
  )
}
```
## Type Definitions

Here are the type definitions used in the `<CustomCommand />` component:

| Name                      | Type                                                                                 | Description                                                        |
| ------------------------- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| `Command`                 | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive>>`                   | Main wrapper component for the command menu.                       |
| `CommandDialog`           | `React.FC<DialogProps>`                                                             | Dialog component for showing the command menu.                     |
| `CommandEmpty`            | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.Empty>>`             | Component displayed when there are no search results.              |
| `CommandGroup`            | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.Group>>`             | Groups related command items together.                             |
| `CommandInput`            | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.Input>>`             | Input field for searching commands or items.                       |
| `CommandItem`             | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.Item>>`              | Represents an individual item in the list.                         |
| `CommandList`             | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.List>>`              | Container for the list of command items.                           |
| `CommandSeparator`        | `React.FC<React.` `ComponentPropsWithoutRef<typeof` `CommandPrimitive.Separator>>`         | Separates different groups of command items.                       |
| `CommandShortcut`         | `React.FC<React.` `HTMLAttributes<HTMLSpanElement>>`                                     | Displays keyboard shortcuts for command items.                     |
| `CommandListGroup`        | `React.FC<CommandListGroupType>`                                                      | Custom component to render grouped command items with options.     |

### CommandListGroupDataType

| Property  | Type                        | Description                                                      |
| --------- | --------------------------- | ---------------------------------------------------------------- |
| `label`   | `string`                    | The label or name of the item, used as the display text.         |
| `element` | `React.ReactElement | string` | The content that will be rendered, can be a React element or a string. |
| `onSelect`| `<T>(arg?: T) => void`      | An optional callback function triggered when the item is selected. |


### CommandListGroupType

| Property       | Type                                 | Description                                                                                  |
| -------------- | ------------------------------------ | -------------------------------------------------------------------------------------------- |
| `data`         | `CommandListGroupDataType[]`         | An array of items (`label`, `element`, `onSelect`) to be displayed within the group.         |
| `group`        | `number[]`                           | An optional array of numbers that define how many items should be in each group.             |
| `groupheading` | `string[]`                           | An optional array of headings for each group.                                                |
| `onSelect`     | `<T extends string>(arg?: T) => void`| An optional callback function triggered when an item is selected.                            |
| `selected`     | `string[]`                           | An array of strings representing the labels of selected items.                               |
| `className`    | `string`                             | An optional string for custom CSS classes.                                                   |
| `checkabble`   | `boolean`                            | An optional boolean to show a checkbox next to each item if set to `true`.                   |
