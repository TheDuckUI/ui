## Features

- **Dynamic Navigation Grouping:** Organize navigation buttons into customizable groups based on specified configurations.
- **Collapsible Navigation:** Supports collapsible sections for a more compact and user-friendly navigation experience.
- **Customizable Buttons:** Easily modify button properties such as variant, icon, and click behavior for tailored interactions.
- **Filtered Props Handling:** Automatically filters out unnecessary props for cleaner and more efficient component rendering.

## Usage

```tsx
import { NavGroup, TooltipProvider } from "@/components/ui"
```

```tsx
const data = [
  {
    title: 'Home',
    route: '/home',
    children: 'Home',
    icon: <Home />,
    label: {
      children: '21',
    },
  },
  {
    title: 'Calendar',
    route: '/calendar',
    children: 'Calendar',
    icon: <Calendar />,
    label: {
      children: '20',
    },
  },
]

export default function App() {
  return (
    <>
      <TooltipProvider>
        <NavGroup<true>
          nav={{
            pathname: '',
            group: group as number[],
            router: {},
            location: {},
            isCollabsed: open,
          }}
          navigationKeys={data}
        />
      </TooltipProvider>
    </>
  )
}
```

### NavGroupProps

The `NavGroupProps` interface defines the properties that can be passed to the `NavGroup` component.

#### Parameters

| Parameter       | Type                                                | Description                                                                                   |
| --------------- | --------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `navigationKeys`| [`ButtonProps[]`](/docs/components/Button#buttonprops)                                     | An array of button configurations for rendering the navigation buttons.                       |

| `...props`      | `React.HTMLAttributes<HTMLDivElement>`              | Additional HTML attributes to apply to the navigation group element.                          |

### NavType

The `NavType` interface defines the configuration for navigation, including whether it is collapsible.

#### Parameters

| Parameter       | Type                                                | Description                                                                                   |
| --------------- | --------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `group`         | `number[]`                                          | An array defining how many buttons should be grouped together.                                |
| `router`        | `any`                                               | Router instance for handling navigation.                                                      |
| `pathname`      | `string`                                            | Current path to highlight active buttons.                                                     |
| `className`     | `string` (optional)                                 | Optional additional class names for styling the navigation group container.                   |
| `isCollabsed`   | `boolean` (optional)                                | Indicates if the navigation section is collapsible.                                           |

### NavCollabsableType

The `NavCollabsableType` type defines whether the navigation can be collapsible or not.

#### Parameters

| Parameter       | Type                                                | Description                                                                                   |
| --------------- | --------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `isCollabsed`   | `boolean` (optional)                                | Indicates if the navigation section is collapsible.                                           |

### NavCollabsedType

The `NavCollabsedType` interface is used when the navigation section is collapsible.

#### Parameters

| Parameter       | Type                                                | Description                                                                                   |
